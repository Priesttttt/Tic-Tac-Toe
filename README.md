# Tic Tac Toe

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript. Play against another player in a clean and responsive interface.

## Running the Project

Open the project folder and run:

```bash
python -m http.server 8000
```

or

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

You can also open `index.html` using the Live Server extension in VS Code.

## Features

* Two-player gameplay
* Win and draw detection
* Winning cell highlighting
* Restart game functionality
* Responsive design
